This is a classic Snake arcade game implementation.

Players control the snake, eat food, and grow longer without colliding.

## 🎮 Gameplay

- Move snake in grid system / 网格移动  
- Eat food to grow / 吃食物增长  
- Avoid collision / 避免碰撞  
- Increase score / 得分系统
